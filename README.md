# Next.js Fintech Dashboard

A mini Next.js project demonstrating core rendering concepts.

## Features
- File-based routing using App Router
- Server-Side Rendering (SSR)
- Static Site Generation (SSG)
- Simple dashboard-style UI

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS

## Pages
- `/` – Home
- `/dashboard` – SSR page
- `/features` – SSG page
